Caesar Cypher

A JavaScript encoding/decoding web app using the Caesar cypher and an HTML user interface. An additional level of security has been added to the cypher through the use of a rolling substitution. This will require the user to input two keys - first key dictates the initial shift, the second key dictates how much that shift changes for each new letter. If the user inputs any characters that are not letters, these characters will be returned in their original form.
